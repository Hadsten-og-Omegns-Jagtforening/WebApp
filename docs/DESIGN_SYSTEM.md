# HOJ Design System

## Core direction

- Practical
- Local
- Warm
- Readable
- Not corporate
- Not SaaS
- Not startup-like

## Colors

- Forest: #1E2A22
- Bone: #F4EFE4
- Brass: #B8893A
- Clay: #C7502E (ONLY for alerts/danger)

## Typography

- Large readable text
- High contrast
- No fancy UI tricks

## Layout rules

### Hero
- Split layout
- Image left
- Content card right
- Light background (Bone)

### Navigation
- Simple
- Clear hierarchy
- No hidden complexity

### Cards
- Light background
- Clear spacing
- No dark-heavy UI

## UX rules

- Assume older users
- No hover-only interactions
- Large clickable areas
- Clear labels

## Forbidden

- Dark SaaS hero layouts
- Minimalist startup UI
- Neon / hacker style
- Over-designed components
