Key Principles

  • Public news pages are server-rendered — Google indexes them fully
  • Next.js middleware redirects unauthenticated requests away from /admin/*
  • Existing CSS design system (colors_and_type.css + kit.css) migrates as-is
  • Existing JSX components (Nav, Footer, cards) migrate to TypeScript React
  • TypeScript throughout — catches bugs early, good for long-term maintenance
